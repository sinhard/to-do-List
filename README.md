# ToDo-List
ToDo list is a simple web application to save Your daily tasks in order not to miss anything

# Features 
New Task
Delete single task
Delete completed tasks
  
# Technologies used:
* HTML
* CSS
* JavaScript 

# Installation 
* Clone repository 
* Open index.html, using a browser.
